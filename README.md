# ThriveHub (Public) — Quick Start

**Your GitHub Pages URL:** `https://DGR125.github.io/thrivehub/`

## Files
- `index.html` — the full app (categories, strategies, live links, likes, dark mode)
- `thrivehub-logo.png` — logo for header/footer (replace with your preferred logo if desired)
- `favicon.png` — browser tab icon
- `ThriveHub_Deployment_Guide.pdf` — polished guide for your team

## Publish on GitHub Pages
1. Create a public repo named `thrivehub` in your GitHub account `DGR125`.
2. Upload all files in this folder.
3. Go to **Settings → Pages** and set **Source** to `main` and **/root`**.
4. Wait ~60 seconds. Your site will be live at `https://DGR125.github.io/thrivehub/`.

## Run locally
Double‑click `index.html` (Chrome/Edge/Firefox).

## Customise
- Replace `thrivehub-logo.png` with your preferred logo file (keep the same name).
- Edit strategies/resources directly in `index.html` using Notepad.

Enjoy! — ThriveHub
